Добро пожаловать в **Flibusta Bot** — ваш персональный помощник для поиска и скачивания книг из библиотеки Флибуста прямо в Telegram!

## 🚀 Начало работы

### Первый запуск

Отправьте боту команду `/start` чтобы начать работу. Бот покажет приветственное сообщение с основными командами и возможностями.

### Основные команды

| Команда | Описание                                                       |
|---------|----------------------------------------------------------------|
| `/start` | Запуск бота и отображение приветствия                          |

| `/pop` | Популярные книги и новинки                                     |

| `/donate` | Поддержать разработчика                                        |

## 🔍 Поиск книг

### Простой поиск

Просто отправьте боту любые слова для поиска. Бот будет искать по **названию книги, авторам, жанрам, сериям и году издания**.

**Примеры:**
```
Лев Толстой война мир
фантастика космос звёзды 2025
Harry Potter
Перельман математика физика
```

### Расширенный поиск с операторами

Для более точного поиска используйте специальные операторы:

#### ➕ Обязательное слово
Используйте `+` перед словом, чтобы оно обязательно было в результатах:

```
+жизнь +замечательных людей
+Распутин -Валентин
```

#### ➖ Исключение слов
Используйте `-` перед словом, чтобы исключить его из результатов:

```
фантастика -ужасы
детектив -криминал
```

#### ⭐ Поиск по частям слов
Используйте `*` для поиска по части слова:

```
математи* задач*
фантаст* роман*
```

#### 🔄 Группировка слов
Используйте скобки `()` для группировки условий:

```
+(эльф гоблин орк гном) +(одинокий злой грустный огромный)
+(путешествия вторжения пришельцев) +(космические времени другие миры)
```

#### 📏 Расстояние между словами
Используйте `@N` после фразы в кавычках для поиска слов на определенном расстоянии:

```
"настоящее время проживает Москве" @5
"умер 1980 году" @5
"воевал получил ранение" @10
```

## 📖 Поиск по аннотациям

Бот может искать по полным текстам описаний книг и биографий авторов. Для этого нужно настроить область поиска.

### Включение поиска по аннотациям

1. Отправьте команду `/set`
2. Выберите **"Область поиска"**
3. Выберите один из вариантов:
   - **по аннотации книг** — поиск по описаниям книг
   - **по аннотациям авторов** — поиск по биографиям авторов

### Примеры запросов для аннотаций книг

```
+(эльфы гномы орки) +(злые одинокие большие грустные)
+автор +создал удивительный волшебный +мир
+русский +полководец Отечественной войны
+сборник +фантастических +(рассказов повестей) +советских -(зарубежных иностранных)
```

### Примеры запросов для аннотаций авторов

```
+награждён +(медалями орденами)
+(родился проживал умер) +Пензенской +губернии
+филолог +переводчик +журналист
```

## 📚 Взаимодействие с результатами поиска

### Формат отображения книг

Каждая книга отображается в виде кнопки со следующим форматом:

**🔵 Название книги (Автор) размер/Жанр/Год**

- **Эмодзи рейтинга** в начале (⚪️🔴🟠🟡🟢🔵)
- Название, автор, размер, жанр и год издания
- Текст автоматически обрезается по ограничениям Telegram

### Просмотр результатов

После поиска бот покажет список найденных книг с кнопками:

- **📥 Скачать** — скачать книгу в выбранном формате
- **📖 О книге** — подробная информация о книге
- **👤 Об авторе** — информация об авторе
- **💬 Отзывы** — отзывы читателей
- **❌ Закрыть** — закрыть информацию

### Навигация по страницам

Если количество книг превышает установленный **Постраничный вывод** (20/40), в клавиатуру добавляются кнопки навигации:

- **◀️ Назад** — предыдущая страница
- **▶️ Вперед** — следующая страница
- **⏪ В начало** — первая страница
- **⏩ В конец** — последняя страница

### Группировка результатов

В настройках (`/set` → "Вывод результатов") можно выбрать способ группировки:

#### По книгам
Прямой список книг с кнопками для скачивания и просмотра информации.

#### По авторам
1. Сначала выводится клавиатура с кнопками авторов
2. Каждая кнопка содержит: **Имя автора (количество книг)**
3. Авторы отсортированы по убыванию количества книг
4. При нажатии на автора:
   - Сообщение подменяется на клавиатуру с книгами этого автора
   - Добавляются кнопки: **👤 Об авторе** и **◀️ Назад к авторам**

#### По сериям
1. Сначала выводится клавиатура с кнопками серий
2. Каждая кнопка содержит: **Название серии (количество книг)**
3. Серии отсортированы по убыванию количества книг
4. При нажатии на серию:
   - Сообщение подменяется на клавиатуру с книгами этой серии
   - Добавляется кнопка: **◀️ Назад к сериям**

## 🏷️ Жанры

Команда `/genres` открывает меню жанров:

1. Выводится клавиатура с **родительскими жанрами**
2. Каждая кнопка содержит: **Название жанра (количество книг)**
3. Жанры отсортированы **по алфавиту**
4. При нажатии на жанр:
   - Выводится сообщение с дочерними жанрами
   - Каждый поджанр отображается как **ссылка на flibusta.is**
   - Рядом указывается количество книг в формате (N)

## ⚙️ Настройки

Команда `/set` открывает меню настроек:

### Доступные настройки

| Настройка | Описание | Варианты |
|-----------|----------|----------|
| **Постраничный вывод** | Количество книг на странице | 20, 40 |
| **Язык книг** | Фильтр по языку | ru, en, de, fr, и др. |
| **Ограничение на размер** | Фильтр по размеру файла | <800K, >800K |
| **Фильтр по рейтингу** | Фильтр по оценкам читателей | ⚪️ Без рейтинга, 🔴 Нечитаемо, 🟠 Плохо, 🟡 Неплохо, 🟢 Хорошо, 🔵 Отлично |
| **Формат скачивания** | Формат файла для скачивания | FB2, MOBI, EPUB |
| **Вывод результатов** | Способ группировки | по книгам, по сериям, по авторам |
| **Область поиска** | Где искать | по основным данным, по аннотациям книг, по аннотациям авторов |

> 💡 **Текущие значения настроек** отображаются в **круглых скобках** на кнопках главного меню настроек.

### Как изменить настройку

1. Отправьте `/set`
2. Выберите нужную настройку
3. Выберите значение из предложенных вариантов
4. Нажмите "Назад" чтобы вернуться в главное меню

### Применение настроек

Настройки **языка, размера и рейтинга** применяются ко **ВСЕМ** поискам:
- Обычный поиск по всем областям
- Популярные книги (`/pop`)
- Новинки (`/pop`)

## 📥 Скачивание книг

### Процесс скачивания

1. Найдите книгу через поиск
2. Нажмите кнопку **"📥 Скачать"**
3. Бот пришлет файл в выбранном формате

### Доступные форматы

- **FB2** — FictionBook (стандартный формат для электронных книг)
- **MOBI** — для устройств Amazon Kindle
- **EPUB** — универсальный формат для большинства читалок

**Важно:** Формат по умолчанию — FB2. Измените в настройках, если нужен другой формат.

## 🎯 Советы по эффективному поиску

### 1. Используйте несколько слов
Бот ищет книги, содержащие **любое** из указанных слов. Чем больше слов, тем точнее результаты.

### 2. Комбинируйте операторы
```
+фантастика +космос -ужасы
```
Найдет фантастику про космос, но без ужасов.

### 3. Ищите по аннотациям для сложных запросов
Хотите найти книги с определенным сюжетом? Используйте поиск по аннотациям:

```
+главный герой +путешествует +во +времени
```

### 4. Фильтруйте по рейтингу
В настройках включите фильтр по рейтингу, чтобы видеть только качественные книги.

### 5. Используйте жанры
Команда `/genres` помогает открыть для себя новые книги в любимых жанрах.

### 6. Настройте группировку
Если ищете все книги автора, используйте **"по авторам"**. Если ищете книги из серии — **"по сериям"**.

## 🆘 Часто задаваемые вопросы

### Как найти книги определенного автора?
Просто введите фамилию автора:
```
Достоевский
Толстой Лев
```

### Как найти книги из определенной серии?
Введите название серии:
```
Гарри Поттер
Властелин колец
```

### Почему не находится книга, которая есть на Флибусте?
Попробуйте:
1. Упростить запрос
2. Использовать часть названия с `*`
3. Проверить настройки языка и фильтров
4. Использовать поиск по аннотациям

### Как скачать книгу в другом формате?
Измените настройку "Формат скачивания книг" в меню `/set`, затем снова нажмите "Скачать".

### Что делать, если бот не отвечает?
1. Проверьте интернет-соединение
2. Подождите несколько минут (бот может быть перегружен)
3. Попробуйте отправить запрос еще раз
4. Обратитесь в поддержку
