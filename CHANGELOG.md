# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]
### Планируется
- Типизация всех функций
- Рефакторинг в классы
- Оптимизация SQL запросов

### Added
- Миграция на cb_ префикс для таблиц Flibusta
- Безопасная стратегия обновления БД

### Changed  
- Все SQL запросы используют cb_libbook вместо libbook
- Все SQL запросы используют cb_libavtor вместо libavtor
- Все SQL запросы используют cb_libgenre вместо libgenre
- И т.д. для всех lib* таблиц

## [1.0.0] - 2025-01-15

### Added
- Полнотекстовый поиск по базе MariaDB
- Поиск по аннотациям книг и авторов
- Поддержка групповых чатов
- Фильтрация по рейтингу книг
- Популярные книги и новинки
- Админ-панель с статистикой
- Донаты через Telegram Stars

### Changed
- Миграция с SQLite на MariaDB для основной БД
- Упрощён интерфейс поиска

### Fixed
- Скачивание книг на всех языках
- Стабильность работы в группах
