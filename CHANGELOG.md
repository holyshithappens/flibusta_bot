# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased]

### Added
- Рефакторинг в Service/Repository/Handler архитектуру
- Структурированное логирование в JSON формате
- Базовые классы BaseMariaDBRepository и BaseSQLiteRepository
- LogsRepository для работы с логами
- StructuredLogger для единообразного логирования
- Аналитические методы для статистики

### Changed  
- Разделение database.py на репозитории
- Логи записываются в БД с типизированными полями
- Исключено логирование действий администратора

## [1.1.0] - 2025-12-14

### Changed
- Миграция на cb_ префикс для таблиц Flibusta
- Безопасная стратегия обновления БД

### Added
- SQL скрипты для миграции и отката
- Документация процесса обновления БД

## [1.0.0] - 2025-01-15

### Added
- Полнотекстовый поиск по базе MariaDB
- Поиск по аннотациям книг и авторов
- Поддержка групповых чатов
- Фильтрация по рейтингу книг
- Популярные книги и новинки
- Админ-панель с статистикой
- Донаты через Telegram Stars

### Changed
- Миграция с SQLite на MariaDB для основной БД
- Упрощён интерфейс поиска

### Fixed
- Скачивание книг на всех языках
- Стабильность работы в группах
